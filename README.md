# CO₂ Emissions Dashboard (Vehicle Analysis)

##  Project Overview

This project presents an interactive dashboard that analyzes **CO₂ emissions produced by different types of vehicles**. The goal is to provide clear insights into emission patterns, helping users understand environmental impact based on vehicle categories, fuel types, and other key factors.

---

## Objectives

* Analyze CO₂ emissions across various vehicle types
* Identify high-emission categories
* Compare fuel efficiency and environmental impact
* Provide actionable insights through data visualization

---

##  Dataset

The dataset includes:

* Vehicle type/category
* Engine size
* Fuel type
* Fuel consumption
* CO₂ emissions (g/km)

---

##  Tools & Technologies

* **Power BI** – Dashboard creation & visualization
* **DAX** – Measures and calculated columns
* **Excel / CSV** – Data preprocessing


---

##  Key Features

*  Interactive visualizations (charts, graphs, KPIs)
*  Filters for vehicle type, fuel type, and emission levels
*  Data bars and matrix representation for comparison
*  Top & bottom emission categories analysis
*  Trend insights for better decision-making
